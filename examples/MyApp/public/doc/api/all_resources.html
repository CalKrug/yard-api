Shirts API

Interfaces

    Create a new shirt

    POST /shirts

    Parameters

    • shirts[color] String Required

      The color of the shirt.

    • shirts[size] String [ S, M, L, XL ] Required

      The size of shirts you wear.

    Example Request

    JSON cURL 
    {
  "shirts": [{
    "color": "red",
    "size": "M"
  }]
}
    curl \
  -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json' \
  -d '{"shirts":[{"color":"red","size":"M"}]}' \
  http://<my_app>/api/shirts

    Example Response:

    {
  "shirts": [{
    "id": "1"
  }]
}

    Return a shirt BETA

    POST /shirts/:shirt_id/return

    Return a shirt, probably 'cause you don't like it.

    Parameters

    • shirt_id Integer Required

      ID of the shirt you want to return.
