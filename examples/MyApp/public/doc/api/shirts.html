    Object Synopses

    Shirt

    A real T-Shirt.

    • id String
      No description provided.
    • color String

      Color of the shirt.

    • size String Required

      Size of the shirt.

    Create a new shirt

    POST /shirts

    Parameters

    • shirts[size] String [ S, M, L, XL ] Required

      The size of shirts you wear.

    • shirts[color] String

      The color of the shirt.

    Example Requests

    JSON cURL 
    {
  "shirts": [{
    "color": "red",
    "size": "M"
  }]
}
    curl \
  -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json' \
  -d '{"shirts":[{"color":"red","size":"M"}]}' \
  http://<my_app>/api/shirts
